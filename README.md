# Bachelorarbeit_2026
Repository zur Bachelorarbeit über den Einfluss der Sampling-Temperatur auf Halluzinationen in RAG-gestützten Large Language Models. Enthalten sind Experiment-Setup, RAG-Pipeline, Evaluationsskripte, Metriken, statistische Auswertungen und Visualisierungen zur Analyse faktischer Zuverlässigkeit.
